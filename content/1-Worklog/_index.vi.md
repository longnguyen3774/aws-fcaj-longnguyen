Đây là nhật ký làm việc (worklog) của tôi trong suốt chương trình thực tập **First Cloud AI Journey** - một chương trình tập trung chuyên sâu vào Cloud và AI trên nền tảng AWS. Mỗi bài viết ghi lại chi tiết các mục tiêu học tập hàng tuần, các dịch vụ đã nghiên cứu, các tác vụ thực hành đã hoàn thành cùng với những bài học kinh nghiệm quý giá rút ra.

Chương trình kéo dài khoảng **3 tháng** (từ ngày 20/04 đến ngày 10/07/2026), bao gồm các chủ đề từ nền tảng đến nâng cao về khởi tạo mạng lõi đám mây, tối ưu hóa lưu kho dữ liệu đa tầng, chuỗi pipeline xử lý dữ liệu lớn (Big Data) và tự động hóa vận hành mô hình học máy (MLOps) phục vụ dự báo thực tế trên AWS.

---

**Tuần 1**: [Kiến Trúc Mạng Bản Địa Đám Mây, Khởi Tạo Phân Phố Topo Amazon VPC & Thiết Lập Cổng NAT Gateway Cho Private Subnet](1.1-Week01/)

**Tuần 2**: [Chiến Lược Phân Quyền Hạn Đặc Quyền Tối Thiểu AWS IAM, Cô Lập Nhóm Bảo Mật Đa Tầng & Kiểm Toán Hệ Thống Bằng VPC Flow Logs](1.2-Week02/)

**Tuần 3**: [Thiết Kế Lược Đồ Quan Hệ Database Giao Dịch & Phân Tích, Định Hình Cấu Trúc Tiền Tố Lưu Trữ Thư Mục S3 & Giám Sát Hạn Mức Ngân Sách CloudWatch](1.3-Week03/)

**Tuần 4**: [Triển Khai Cụm Máy Chủ Tính Toán Storefront Trên Nhiều Vùng Sẵn Sàng (AZs), Thiết Lập Phân Tầng RESTful API Riêng Tư & Chuẩn Hóa Môi Trường Runtime](1.4-Week04/)

**Tuần 5**: [Tối Ưu Hóa Truyền Tải Tài Nguyên Toàn Cầu Qua Mạng Lưới CloudFront CDN, Điều Phối Traffic Với External ALB & Che Giấu API Với Internal ALB](1.5-Week05/)

**Tuần 6**: [Cô Lập Hệ Thống Database Đôi Amazon RDS PostgreSQL, Kích Hoạt Sao Lưu Tự Động, Cấu Hính DB Subnet Groups & Kiểm Thử Kết Nối Kết Cuối](1.6-Week06/)

**Tuần 7**: [Kiến Trúc Nạp Nhập Dữ Liệu, Đồng Bộ Thư Mục Dữ Liệu Qua AWS Glue Data Catalog & Trích Xuất Dữ Liệu Gốc Tiết Kiệm Chi Phí Với Glue Python Shell](1.7-Week07/)

**Tuần 8**: [Phát Triển Thuật Toán Biến Đổi Dữ Liệu Lớn, Xử Lý Đặc Trưng Chuỗi Thời Gian Định Dạng Phân Tán Bằng PySpark Window Functions & Kiểm Thử Logic](1.8-Week08/)

**Tuần 9**: [Triển Khai Chuỗi Pipeline PySpark ETL Lên Môi Trường Quản Trị AWS Glue, Tinh Chỉnh Cấu Hình Cluster DPU & Ghi Kết Quả Đồng Bộ Về Database Phân Tích](1.9-Week09/)

**Tuần 10**: [Khởi Tạo Máy Chủ Máy Chủ Tính Toán Huấn Luyện ML Chuyên Dụng, Tối Ưu Hóa Vòng Lặp Thuật Toán Hồi Quy & Tự Động Đóng Gói Mô Hình Thành Phẩm Lên S3](1.10-Week10/)

**Tuần 11**: [Triển Khai Phục Vụ Mô Hình Không Máy Chủ (Serverless Inference) Với AWS Lambda, Điều Hướng API Gateway Linh Hoạt & Xác Thực Độ Trễ Điểm Kết Cuối](1.11-Week11/)

**Tuần 12**: [Tự Động Hóa Toàn Diện Hệ Thống Dựa Trên Sự Kiện Bằng Amazon EventBridge Scheduler, Trực Quan Hóa Hạ Tầng CloudWatch Dashboard & Cảnh Báo Khẩn Cấp AWS SNS](1.12-Week12/)
